# Security Policy

## Reporting a Vulnerability

We take the security of Viramah seriously. If you believe you have found a security vulnerability in our platform, please report it to us responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to [security@viramah.com](mailto:team@viramah.com).

### Our Commitment

If you report a vulnerability to us, we will:

1. Acknowledge your report within 2 business days.
2. Investigate the issue and provide a fix or mitigation plan as quickly as possible.
3. Keep you informed of our progress.
4. Give credit to the reporter (if desired) once the issue is resolved.

### Scope

This policy applies to:
- The Viramah Web Application (`viramah.com`)
- Our API endpoints
- Our production infrastructure

Thank you for helping keep Viramah safe.
